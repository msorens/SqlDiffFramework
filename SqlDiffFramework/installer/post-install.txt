SqlDiffFramework is designed for the enterprise! Check with your system administrator for how to:
>> Point to the SqlDiffFramework repository (if any) so you will be notified of updates.
>> Load a set of of default database connections for your environment.

and review the user guide at http://SqlDiffFramework.codeplex.com/documentation.
